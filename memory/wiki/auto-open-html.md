更新完 HTML 檔案後自動用瀏覽器打開預覽，不要再問「要我打開嗎？」。

**Why:** 使用者每次都會想看結果，問一次是多餘的。
**How to apply:** 任何 `docs/*.html` 或類似的視覺化檔案寫入/更新後，直接 `start "" "path"` 打開。

這個規則也適用於 [[whiteboard-standard]] 的白板 HTML 更新。
