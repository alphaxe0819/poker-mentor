# 賽事合作策略

最後更新：2026-04-13

## 核心設計

**月榜代幣 → 兌換賽事入場資格**（不是點數直接換門票）

流程：MTT 模擬 → 賺積分 → 月榜排行 → 前段獲代幣 → 代幣兌換賽事資格

### 台灣市場（第一階段）
- 合作對象：**WinWin**（台灣撲克賽事主辦方）
- 機制：用戶參加 MTT 模擬累積積分 → 月榜前段獲代幣 → 兌換 WinWin 賽事入場
- 代幣有實體價值，對認真玩家吸引力遠高於純數位獎勵
- 驅動用戶持續參賽提升平台黏性

### 海外市場
- **不找協會，找賽事平台合作**（實體或線上）
- 每進入新市場，談當地一個賽事平台
- 用「代幣兌換賽事資格」作為切入合作條件
- 優勢：比純廣告合作說服成本低，雙方互導流量

## 商業邏輯

| 角色 | 獲益 |
|------|------|
| Poker Goal | 驅動用戶持續參賽，提升黏性與留存 |
| WinWin / 賽事平台 | 免費獲得 Poker Goal 用戶流量，報名人數增加 |
| 用戶 | 練習有回報（月榜代幣 = 真實賽事資格），學習動機更強 |
