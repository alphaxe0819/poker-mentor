為什麼應該 3-Bet？

AKo 在 BTN vs UTG 加注的情境是高價值牌，在 SRP（single raised pot）即便被 call 也能頻繁靠強 top pair + 好 kicker 贏下底池。

Call 會錯過壓縮範圍的價值：3-Bet 後 UTG 需面對更窄的 continue range，你拿到主動權 + 更好的翻牌圈底池幾何。

僅建議在特別深籌（150BB+）或 UTG 是極 tight 玩家時降頻 call。
