AI 教練行為準則 = 教練設定 + 系統干預

Coach Behavior = Creator Settings + System Override

🎨 教練設定（個性層）

  • System Prompt：教練人設背景故事
  • 語氣風格：毒舌 / 鼓勵 / 分析型
  • 教學方法：蘇格拉底式提問 / 直接給答案
  • GTO 偏好：純 GTO / Exploit 傾向 / ICM 專家
  • 回應長度：簡潔 / 詳細解說
  • 招牌用語：每位教練獨有的口頭禪

🛡️ 系統干預（安全層）

  • GTO 正確性護欄：確保建議不偏離 GTO 範圍
  • 禁止話題：不討論線上作弊、外掛工具
  • 點數消費控制：每則訊息扣點，餘額不足提醒
  • 內容審核：過濾不當言論
  • 上下文長度：限制對話歷史 token 數
  • 角色邊界：教練不能被 jailbreak 成非撲克角色

模擬比賽練習模式

Tournament Simulator
比賽模式 人數 入場費 特色
快速 SNG 9人 50 點 15分鐘速戰
MTT 模擬 27人 100 點 盲注遞增 + 泡沫期

核心機制

  • 虛擬籌碼：獨立於點數，比賽內使用
  • 盲注結構：自動遞增，模擬真實錦標賽
  • AI 對手：用 GTO 範圍 + 隨機化模擬決策
  • ICM 計算：泡沫期自動調整策略建議
  • 即時排名：比賽中顯示籌碼排名
  • 賽後分析：教練回顧關鍵決策點

獎勵機制

  • 🥇 第1名：入場費 x5 點數返還
  • 🥈 第2名：入場費 x3
  • 🥉 第3名：入場費 x1.5

永久記憶系統

Persistent Memory (RAG)
  • 學習歷程：記住用戶做過的課題和正確率
  • 弱點追蹤：記住用戶常犯錯的位置/場景
  • 風格偏好：記住用戶偏好的教練和訓練模式
  • 對話上下文：跨 session 記住之前的教學進度

技術實現

  • 向量資料庫：Supabase pgvector 儲存記憶嵌入
  • 記憶摘要：每次對話結束自動生成摘要
  • 相關性檢索：RAG 檢索最相關的歷史記憶
  • 記憶衰減：舊記憶權重自動降低
⚠️ Pro 訂閱專屬功能，免費/基礎用戶每次對話獨立
